# 内村鑑三

内村 鑑三（うちむら かんぞう、万延2年2月13日（1861年3月23日）- 昭和5年（1930年）3月28日[1]）は、日本のキリスト教思想家・文学者・伝道者・聖書学者。福音主義信仰と時事社会批判に基づく日本独自のいわゆる無教会主義を唱えた。「代表的日本人」の著者でもある。
