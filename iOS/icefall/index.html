Readme - Icefall

The app, 'Icefall: Icecast Stream Client', provides you the ability to broadcast the sounds recorded by the microphone of iOS device, to the Icecast 2 online radio server.

You need to own an Icecast 2 server first. The function is very simple: configure your server and account information, then open the switch to broadcast.
