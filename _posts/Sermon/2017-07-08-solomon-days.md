# 所羅門年日考

所羅門的一生歲數的時間節點不詳。

之前整理過這麼個表格。

| 年份 | 事件 |
|------|------|
| 2973 | 耶西 生 大卫 |
| 3003 | 扫罗 阵亡 |
| | 大卫登基的时候年三十岁、在位四十年（这里是在希伯仑作犹大王七年零六个月） |
| 3010 | 大卫 在耶路撒冷作以色列和犹大王三十三年。 |
| 3042 | 所罗门 生 罗波安 |
| 3043 | 所罗门 即位 |
| | 大卫 死 |
| 3047 | 二月初二日 所罗门开工建立圣殿（列王记上6:1,注意实际上要在480年之上再加上士师记中失去的114年，合计594年） |
| | 所罗门在位、第四年、西弗月、立了耶和华殿的根基。 |
| 3054 | 所罗门建殿完毕 到十一年、布勒月、就是八月、殿和一切属殿的、都按着样式造成。他建殿的工夫、共有七年。 |
| 3067 | 所罗门宫室营毕 以他念月、就是七月、在节前、献殿 |
| 3083 | 所罗门 死 |
| | 耶罗波安 在示剑登基为以色列王 |

| | 八月十五日 耶罗波安在伯特利牛犊处献祭，犹大神人至，预言约西亚之作为 |
| 3088 | 罗波安王第五年、埃及王示撒上来攻取耶路撒冷（好像埃及換了個新王朝） |
| 3100 | 罗波安 死 |
| | 亚比央 登基做犹大王 |

按照上面的表格的話，大衛進入耶路撒冷是3010年，而所羅門生下羅波安是3042年（合大衛69歲），次年所羅門登基。因此猶太人傳統認為，所羅門登基時至少12歲，考慮到都生娃了，再小也不合適。如果大衛生所羅門之後，偷吃拔示巴的咒詛（刀劍不離家）才發動的話，那麼暗嫩捕食他媽時所羅門已經出生，然後到押沙龍被約押報了燒田之仇殺掉，已經過了11年。之後還有三年的饑荒。這麼說來，所羅門登基少說也15歲了。然後關於所羅門的年紀，聖經完全不記，就說了個作王四十年。估摸著壞事干太多沒必要記了。

羅波安之母為亞捫人拿瑪。在大衛還活著的時候，公然捕食亞捫人的所羅門也是厲害，各種緣由，真是一時也說不盡啊。

順帶一提，羅波安5歲的時候聖殿開工，12歲聖殿造完，25歲的時候王宮造完，從小滿腦子都是大工程，然後又是各種繁華，苦苦等到所羅門死了，終於自己也能搞大工程了，卻不知道百姓被大衛和所羅門的基建搞疲敝了。這真是可悲。越是興旺的外表，越需要一個求心力，當所羅門在位的時候，百姓迫於其智慧和聲望，當所羅門一死，沒人知道羅波安是不是個好東西。畢竟所羅門在傳道書里第二章也感歎，嗟乎，留遺產給後人是大患啊。
